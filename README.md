# nico-resident

nico-resident is a support tools for [Nicovideo's  ranking ](http://www.nicovideo.jp/ranking).


![]('./gif/intro.gif')

## Features
- [x] List of rankings.
- [x] Period filters.
- [x] Category filters.
- [ ] Manage user's history.
- [ ] Manage favorite.
- [ ] Auto check rankings.

## Development
```
$ git clone git@github.com:abouthiroppy/nico-resident.git
$ npm install
$ npm install -g electron react-tools
$ npm start
```
